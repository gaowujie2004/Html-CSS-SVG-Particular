最后div高度有个缝隙. 这个缝隙是 struct 也就是 幽灵空白节点, 这个东西就是 vercital-align 的参考系了
行盒子, 的高度 怎么计算?
随着 div font-size 变大, 间隙也随着变大

幽灵空白节点 - 就是 strut. strut 就是继承父元素的属性. 形成一个 width:0 的 inline box
默认的对齐是 baseline
inline box的高度: 内容区高度 + 半间距 = line-height

计算:
strut: (font-size/em size)*(win ascent + win descent) + line gap 微软雅黑 line gap: 0
